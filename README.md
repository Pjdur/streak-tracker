# Streak Tracker

A coding streak tracker that helps you stay motivated. It tracks your coding streaks and shows a weekly line and gives badges when you hit milestones. It's simple to use and makes coding feel more fun.

## Installation

Clone the repository and install with Cargo:

```
git clone https://github.com/Pjdur/streak-tracker.git
cd streak-tracker
cargo install --path .
```

## Usage

```bash
streak-tracker # if run without arguments, checks in
streak-tracker checkin # checks in today
streak-tracker status # shows your status stats, i.e. streak, line, badges
streak-tracker week # shows your weekly streak
streak-tracker history # shows your checkin history
```
